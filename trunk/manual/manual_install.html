Install

ゴーストは一般的にnarという拡張子のアーカイブで配布される。
ただし、nar自体はzipと同じなのでサーバによってはzipで配布する場合もある。
lzhでも解凍DLLがあればインストール可能ではあるがよほどの理由が無い場合は使用しない。


nar
install.txtを含んだzip。

なー。

install.txtを含んだゴーストフォルダを本体にD&Dすることによって作成できる。
手動でzip圧縮しリネームしたものと同等。
手動でzip圧縮する際はフォルダを掘り下げてはならない。

ゴーストを配布する際にゴーストとシェルのフォルダにあるprofileフォルダを含めてはならないので注意する事。
ネットワーク更新でも同じことがいえる。

install.txt
インストールするべきフォルダなどの情報を記載するテキスト。
このファイルがなければD&Dでインストールすることは不可能。


developer_options.txt
SSP専用。

ghost/master/savedata.txt,nonar
shell/old/,nonar

上記のような記述のテキストをフォルダに置くとupdates2.dauやnarアーカイブ作成時に
指定されたファイルが含まれない。

指定できるオプション
noupdate updates2.dau/updates.txtに含めない。
nonar NARアーカイブに含めない。

両方除外する場合。
ghost/master/savedata,nonar,noupdate

なお、profileフォルダなどは指定しなくても自動的に除外される。
