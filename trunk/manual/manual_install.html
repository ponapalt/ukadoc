Install

※初心者の方は「配布用ファイルを作る」のページも参照のこと。

ゴーストは一般的にnarという拡張子のアーカイブで配布される。

narはzipの拡張子を変えただけのもので、実体はzipなので、サーバによってはzipで配布する場合もある。

lzhでも解凍DLLがあればインストール可能ではあるがよほどの理由が無い場合は使用しない。

nar

install.txtを含んだzip。

Nanika ARchiveの略。

なー。

install.txtを含んだゴーストフォルダを本体にD&Dすることによって作成できる。

手動でzip圧縮しリネームしたものと同等。

手動でzip圧縮する際はフォルダを掘り下げてはならない。

ゴーストを配布する際にゴーストとシェルのフォルダにあるprofileフォルダや、
セーブデータ等を含めないように注意すること。

ネットワーク更新でも同じことがいえる。

home
　+-ghost
　　　+-myghost
　　　　　+-ghost
　　　　　+-shell
　　　　　+-install.txt
　　　　　+-readme.txt
　　　　　+-thumbnail.png
　　　　　+-developer_options.txt (SSPのみ)

install.txt

インストールするべきフォルダなどの情報を記載するテキスト。 このファイルがなければD&Dでインストールすることは不可能。しない。

developer_options.txt

SSP専用。

ghost/master/savedata.txt,nonar
shell/old/,nonar

上記のような記述をしてフォルダに置くとupdates2.dauやnarアーカイブ作成時に指定されたファイルが含まれない。

指定できるオプション

noupdate　//updates2.dau/updates.txtに含めない。
nonar　//NARアーカイブに含めない。

両方除外する場合。

ghost/master/savedata,nonar,noupdate

なお、以下のフォルダやファイルは指定しなくても自動的に除外される


ファイル

  • desktop.ini
  • thumbs.db
  • folder.htt
  • mscreate.dir
  • .DS_Store
  • _CATALOG.VIX

ディレクトリ

  • profile
  • var
  • __MACOSX
  • XtraStuf.mac
