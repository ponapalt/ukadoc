Install設定
■Index ■Ghost ■Shell ■Balloon ■Install ■SAKURA script ■SHIORI Event ■SHIORI Resource
install.txt(ゴースト、バルーン、プラグイン、ヘッドラインセンサ)
S,C charset,文字コード 表示する文字コード。Shift_JIS推奨。
M,S,C name,オブジェクト名 そのオブジェクトの名前。
M,S,C type,種別 ファイルセットの種別。
ゴーストの場合はghost。
バルーンの場合はballoon。
プラグインの場合はplugin。
ヘッドラインセンサの場合はheadline。
複数パッケージの場合はpackage。(SSPのみ)
M,S,C directory,ディレクトリ名 そのオブジェクトのディレクトリ名。半角英数推奨。
M,S,C refresh,数値 数値に1をセットすると同じディレクトリのファイルを全て消去してからインストールする。
M,S,C refreshundeletemask,ファイル名1;ファイル名2... refreshの消去から除外するファイル名。
複数ある場合はコロンで区切る。
install.txt(ゴーストwithバルーン)
S,C charset,文字コード 表示する文字コード。Shift_JIS推奨。
M,S,C name,オブジェクト名 そのオブジェクトの名前。
M,S,C type,種別 ファイルセットの種別。ゴーストの場合はghost。
S,C directory,ディレクトリ名 そのオブジェクトのゴースト側のディレクトリ名。半角英数推奨。
M,S,C balloon.directory,ディレクトリ名 そのオブジェクトのバルーン側のディレクトリ名。半角英数推奨。
M,S,C refresh,数値 数値に1をセットすると同じディレクトリのファイルを全て消去してからインストールする。
M,S,C refreshundeletemask,ファイル名1;ファイル名2... refreshの消去から除外するファイル名。
複数ある場合はコロンで区切る。
install.txt(ゴーストへの追加ファイル)
M,S,C name,オブジェクト名 そのオブジェクトの名前。
M,S,C type,種別 ファイルセットの種別。追加ファイルの場合はsupplement。
M,S,C accept,ゴースト名 受け取るゴーストの名前。
S,C charset,文字コード 表示する文字コード。Shift_JIS推奨。
install.txt(追加シェル)
S,C charset,文字コード 表示する文字コード。Shift_JIS推奨。
M,S,C name,オブジェクト名 そのオブジェクトの名前。
M,S,C type,種別 ファイルセットの種別。追加シェルの場合はshell。
M,S,C accept,ゴースト名 受け取るゴーストの名前。
M,S,C directory,ディレクトリ名 シェルのディレクトリ名。半角英数推奨。
